# Items in Castle Ravenloft - Catacombs

## Strahd's Coffin (K86)
- **Item ID:** strahds_coffin
- **Description:** Black lacquered wood, silver fittings, red velvet interior
- **Properties:** Strahd's true resting place. Destroying it prevents regeneration.
- **Destruction:** 100 HP, AC 15. If destroyed, Strahd cannot regenerate here (must find new resting place or be destroyed permanently).

## Wooden Stakes
- **Location:** Scattered in alcoves (ancient vampire-hunting tools)
- **Count:** 1d4 stakes
- **Properties:** If driven through Strahd's heart while in coffin, instant kill (no save)
